Idea

Using a real life problem encountered as part of one of the team member's job, the group decided to tackle a problem with a practical use. They decided to try to crop a background out of an image to be used at the State of Iowa's kiosks.

Research

The team researched previous approaches to try to determine the problems other teams had encountered in the past.

Results
  • Background Subtraction

    Background subtraction algorithm using MOG2 background subtraction in OpenCV.

  • Facial Recogition

    Addition of facial recognition using Haar Cascades.

  • Second Background Subtraction

    Custom background subtration algorithm using weighted averages of frames.

  • Grab Cut

    Using GrabCut with no user input.
